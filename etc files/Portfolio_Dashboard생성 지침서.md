# 📊 금융 포트폴리오 대시보드 제작 지침서 (Portfolio Dashboard Guide)

이 지침서는 전문가급 금융 포트폴리오 대시보드 프로젝트의 핵심 사양과 구현 로직을 기록하여, 추후 유지보수 및 기능 확장을 위한 가이드로 활용됩니다.

## 1. 프로젝트 개요
*   **목적**: 엑셀(XLSX) 데이터를 기반으로 투자 비중, 배당 현황, 자산 성장을 시각화하는 다크 모드 대시보드.
*   **기술 스택**: 
    *   **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
    *   **Library**: [Chart.js](https://www.chartjs.org/) (시각화), [SheetJS](https://sheetjs.com/) (엑셀 파싱)
    *   **Backend**: Python Flask (`sync_server.py`) - 주가 및 환율 실시간 동기화.

## 2. 주요 탭별 기능 및 데이터 매칭
### ① 투자비중 (Investment Allocation)
*   **기능**: 계좌별/종목별/분류별 투자 비중 확인 및 계좌합 비교.
*   **데이터 소스**: `'종목현황'` 시트
*   **핵심 로직**:
    *   상단에 **계좌합 1, 2** 비교 카드를 배치하여 체크박스로 동적 합산 가능.
    *   차트 범례는 도넛 차트에서 뻗어 나오는 **Callout 선**으로 표시 (시인성 극대화).
    *   수익(+)은 초록색, 손실(-)은 빨간색으로 통일.

### ② 배당현황 (Dividend Analysis)
*   **기능**: 연도별/월별 배당금 추이 분석 및 실시간 환율 환산.
*   **데이터 소스**: `'배당내역'` 시트 (고정)
*   **매핑 구조**:
    *   **4번 열(Index 4)**: 원화배당금 (₩)
    *   **5번 열(Index 5)**: 외화배당금 ($)
*   **핵심 로직**: 
    *   외화배당금이 존재할 경우 대시보드 상단의 실시간 환율(`usdKrw`)을 곱해 자동으로 원화 합산.
    *   **그룹형 막대 차트**를 통해 연도별 월 배당금을 비교.

### ③ 누적수익률 (Cumulative Return)
*   **기능**: 원금(입금액) 대비 총 자산 성장 추이 시각화.
*   **데이터 소스**: `계좌내역1~10`, `입금내역` 시트
*   **매핑 구조**:
    *   **A열(0)**: 일자 / **H열(7)**: 누적입금액 / **J열(9)**: 평가금액(잔액)
*   **시각화**: 파란색(원금)과 빨간색(자산)의 **영역 차트(Area Chart)**로 성과 비교.

## 3. UI/UX 디자인 원칙
*   **테마**: 현대적인 금융 플랫폼 느낌의 **다크 모드**.
*   **공간 활용**: 계좌 선택 체크박스를 **2열 그리드**로 배치하여 수직 공간 절약.
*   **상호작용**: 탭 전환 시 차트 리사이징 및 데이터 재렌더링 자동화.
*   **시인성**: 어두운 배경에 맞춰 폰트 색상을 최적화하고, 차트 영역 겹침 현상을 `z-index` 및 `transparent header`로 해결.

## 4. 유지보수 가이드
*   **새 계좌 추가**: `'0.인덱스'` 시트에 계좌를 추가하고, 이에 대응하는 `'계좌내역X'` 시트를 생성하면 자동으로 대시보드에 반영됨.
*   **환율/주가 갱신**: 대시보드 상단의 '동기화' 버튼 클릭 시 `sync_server.py`를 통해 최신 정보를 가져옴.
*   **차트 오류 발생 시**: `app.js` 내의 `renderDividendTab()` 또는 `renderCumulativeTab()` 함수의 데이터 파싱 인덱스(Index)가 엑셀 템플릿과 일치하는지 확인.
